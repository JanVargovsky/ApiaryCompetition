# Apiary Competition
Quadient/Apiary competition.

# Requiments
- .NET Core SDK 2.0

# How to run
- navigate to the ApiaryCompetition project's folder
- run **dotnet build -c release**
- navigate to the bin\release\netcoreapp2.0 folder
- run **dotnet ApiaryCompetition.dll**

For CLI help pass **--help** argument.
